# Domainly.ai Frontend

A beautiful and premium multi-domain AI assistant web interface built with React, Vite, and CSS. It supports dark mode toggling, real-time message streaming, and switching focus across multiple expert domains.

## Setup Instructions

1. **Prerequisites**: Make sure you have Node.js (v18+) installed.
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```

The application will be accessible at `http://localhost:5173`.

## Commands

- `npm run dev`: Starts Vite dev server.
- `npm run build`: Compiles production-ready bundle.
- `npm run lint`: Runs ESLint check.
- `npm run preview`: Previews the production build locally.
